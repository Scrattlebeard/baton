# Portrait references (demo)

Reference images generated while researching image-gen options for Baton's
(v1-out-of-scope) portrait feature. `drowned-stacks-refs.zip` holds four
player-safe concept refs for the *Drowned Stacks* story — a house-style
establishing shot, a character portrait, and a crew shot — produced via the
**Pollinations** free API (Flux, no key). The zip's own README documents the
prompts, seeds, and shared style spine, and includes the generator script.

This is a worked example of "generate on-register fantasy refs from a free API,"
not a framework dependency.
